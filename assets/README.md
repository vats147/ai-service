# AI Agent Desktop Assets

This directory contains the application assets:

- icon.png - Main application icon (512x512)
- tray-icon.png - System tray icon (16x16, 32x32)
- logo.png - Application logo

## Icon Requirements

### Application Icon (icon.png)
- Size: 512x512 pixels
- Format: PNG
- Used for: App launcher, dock, taskbar

### Tray Icon (tray-icon.png) 
- Size: 16x16 and 32x32 pixels (or vector SVG)
- Format: PNG or ICO
- Used for: System tray/notification area
- Should be monochrome or simple design for visibility

### Logo (logo.png)
- Size: 256x256 pixels
- Format: PNG with transparency
- Used for: About dialog, loading screen

## Placeholder Icons

The current icons are emoji-based placeholders. For production:
1. Replace with professional vector-based icons
2. Ensure proper sizing for each platform
3. Follow platform design guidelines (Material Design for Linux, Human Interface Guidelines for macOS, Fluent Design for Windows)
